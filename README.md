# ✨ Super Mario Kawaii Edition ✨

Game platformer berbasis web yang terinspirasi dari Super Mario, dikembangkan dengan estetika "Kawaii" menggunakan HTML5 Canvas dan JavaScript murni. Proyek ini dibuat untuk mendukung pengalaman bermain di perangkat desktop maupun mobile (Touch Controls).

## 🌟 Fitur Utama
- **Estetika Kawaii:** Visual bertema pastel yang ceria dengan karakter dan musuh yang lucu.
- **Dukungan Multi-Platform:** Kontrol responsif menggunakan Keyboard (Desktop) dan Touch Buttons (Mobile).
- **Mekanik Gameplay:** 
  - Double Jump (Lompat Ganda).
  - Power-up (Menjadi besar).
  - Beragam tipe musuh (Darat dan Udara).
  - 5 Level unik (Sunny Garden hingga Dream World).
- **Audio Web API:** Efek suara retro dan latar musik yang dihasilkan secara prosedural tanpa file audio eksternal.

## 🕹️ Cara Bermain

### **Desktop (Keyboard)**
- **Panah Kiri/Kanan:** Berjalan.
- **Spasi / Panah Atas:** Lompat (Tekan 2x untuk Lompat Ganda).
- **Tombol P / Esc:** Jeda (Pause) permainan.

### **Mobile (Touch)**
- **Tombol Kiri/Kanan:** Navigasi arah.
- **Tombol Biru (Atas):** Lompat / Lompat Ganda.

## 🛠️ Teknologi yang Digunakan
- **Bahasa:** JavaScript (ES6+)
- **Rendering:** HTML5 Canvas API
- **Audio:** Web Audio API (Oskilator Prosedural)
- **Deployment:** GitHub Pages
